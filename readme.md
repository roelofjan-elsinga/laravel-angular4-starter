## Install the application
- git clone git@github.com:roelofjan-elsinga/euroboatcharter.git
- cd euroboatcharter
- composer install
- cd public && npm install

## Run the application
- Active development: php artisan dev:angular
- Watch production build: php artisan watch:angular
- Production build: php artisan build:angular
